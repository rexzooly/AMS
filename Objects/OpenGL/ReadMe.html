OpenGL Plugin

Please Read Before Use

Note : This plugin is not an OpenGL wrapper , This plugin is a Canvas for OpenGL and you must use a wrapper to draw to this canvas


first , to make an application with this plugin you need minimum level of OpenGL knowledge
once you have that knowledge , you will see that opengl is not control that shows drawings
you need to provide a surface and a window thread to draw something over a window
(you can also draw to a bitmap but this is far away to our subject)

this plugin does all difficult initializations and other tasks , it also provides events that you will need when making animations ,and interactive drawings
you only need to call functions from OpenGL libraries to draw something

in the example sample luagl and its helper luaglu was used as OpenGL wrapper
for AMS users those two modules are very useful
for the function reference you can visit its web site
http://luagl.sourceforge.net/

but it does not mean that you have to use those modules if you can call functions from OpenGL libraries with another way then this plugin may continue working as a canvas

this plugin has two actions , and they are all you need for a Canvas
you can also use multiple instances on a page

so when you want to update a canvas you only need to use those two actions

OpenGL.MakeCurrent("Plugin1"); -- redirect OpenGl Library to our object
--your opengl drawing functions with any way
OpenGL.SwapBuffers("Plugin1"); -- draw buffered data to canvas
you also have to respond On Draw event
because plugin will clear everything over it when a window redaw happens
