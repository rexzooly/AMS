Dependencies Of PieChart Object Plugin

Gdi Plus :

This plugin is linked against the GdiPlus library . The GdiPlus library is a standard component of Windows XP and newer
versions of Windows operating systems. Xp ,Vista , Windows 7 and server versions

If this plugin will work under Windows XP or newer versions of Windows operating systems then there is no dependency
because GdiPlus library is build-in

If target operating system is Windows 2000 then an instance of GdiPlus.dll must be placed in to application root
because Windows 2000 has not GdiPlus.dll by default , and it is not a standard component of Windows 2000

For exaple your application path like below

C:\Program Files\My Company\AutoRun.exe

An instance of GdiPlus.dll should be placed like below

C:\Program Files\My Company\GdiPlus.dll

Simply , it should be placed in the same folder with your application (exe)

Get GdiPlus.dll :

An instance of (redistributable) GdiPlus.dll can be downloaded from here

also you can find other copies by using your favourite search engine
