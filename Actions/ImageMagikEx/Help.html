ImageMagikEx.RotateSquare


(boolean) ImageMagikEx.RotateSquare ( 

(string) imObjectName = "Image1",

(string) baseImage = "AutoPlay\\Images\\rotate.png",

(number) Angle = 0,

(number) Direction = 1,

(number) TimeToWait = 20,

(boolean) useRunNotShellEx = true,

(boolean) loadRenderedImage = false )

Description

RotateSquare: Rotate a .png image, with transparency, Angle and Direction options

Parameters

imObjectName

(string) AMS: Object Image Label ID which contain the displayed image file

baseImage

(string) The full initial image path which will be rotated but never modified.

Angle

(number) The rotation Angle which will be applied to the BaseImage.

Direction

(number) The rotation direction can be clock wise or reverse clock wise. ( use 1 ot -1 )

TimeToWait

(number) The time that application will spend before load the rotated image into his Object Container.

useRunNotShellEx

(boolean) Use File.Run command instead Shell.execute ( With Shell.Execute, extension: .ƒct need to be registered and then computor need to be restarted )

loadRenderedImage

(boolean) If set to true the rotated image will be loaded into Object Image (ObjectName)


Return

This action returns a (boolean) value.

ImageMagikEx.CreateNiceTitle


(boolean) ImageMagikEx.CreateNiceTitle ( 

(string) imObjectName = "Image1",

(string) imSize = "680x180",

(boolean) transparent = true,

(string) imFont = "Comic-Sans-MS",

(number) imPtSize = 72,

(string) imShadowMove = "25,85",

(string) imShadowText = "ImageMagikEx",

(number) imBlurSigma = 6,

(string) imFillColor = "#FF8040",

(string) imStrokeColor = "#2080FF",

(string) imTitlePosition = "20,90",

(string) imText = "ImageMagikEx",

(string) outputImageFilename = "title.png",

(boolean) LoadRenderedImage = true,

(boolean) useRunNotShellEx = false,

(number) imStrokeWidth = 4 )

Description

This action build a title logo image which can be loaded into an object image

Parameters

imObjectName

(string) AMS: Object Image Label ID which contain the rendered title image file

imSize

(string) The size of rendered image title

transparent

(boolean) This argument must be always true but its fixed in the function ( it will be variable may be in next update of this function )

imFont

(string) The font that will be used to render the title

imPtSize

(number) The font size [>24,1000<]

imShadowMove

(string) The position of the shadow title (X,Y)

imShadowText

(string) The shadow text effect, usually same as the title text

imBlurSigma

(number) The argument is th eintensity blur effect on the text

imFillColor

(string) The title text fill color, an hexadecimal color ( eg: #FF8020 )

imStrokeColor

(string) The title text Stroke color, an hexadecimal color ( eg: #FF8020 )

imTitlePosition

(string) The title position (X,Y) - On the global size of the image (imSize)

imText

(string) THE TITLE TEXT

outputImageFilename

(string) The rendered image filename (Filename.Extension)

LoadRenderedImage

(boolean) If set to true the rendered title image will be loaded into Object Image (ObjectName)

useRunNotShellEx

(boolean) This action if set to true, use File.Run command instead Shell.Execute

imStrokeWidth

(number) The stroke text width


Return

This action returns a (boolean) value.


ImageMagikEx.CropAt

(string) imObjectName = "Image1",

(string) baseImage = "AutoPlay\\Images\\temp.png",

(number) baseWidth = 300,

(number) baseHeight = 300,

(number) topHorizontal = 0,

(number) leftVertical = 0,

(number) bottomHorizontal = 0,

(number) rightVertical = 0,

(string) outputImageFilename = "crop.png",

(boolean) loadRenderedImage = true,

(boolean) useRunNotShellEx = false )

Description

This action crop an external rectangle region on a picture (image)

Parameters

imObjectName

(string) AMS: Object Image Label ID which contain the rendered title image file

baseImage

(string) The full path to the image that will be croped (recommended use: in root of AutoPlay\\Images or in a folder in that folder, if not use a full path as: c:\\myfolder\\...\\otherfolder... )

baseWidth

(number) This argument its the width of the final croped image

baseHeight

(number) This argument its the height of the final croped image

topHorizontal

(number) This argument is the top horizontal part on the image that will be croped

leftVertical

(number) This argument is the left vertical part on the image that will be croped

bottomHorizontal

(number) This argument is the bottom horizontal part on the image that will be croped

rightVertical

(number) This argument is the right vertical part on the image that will be croped

outputImageFilename

(string) The rendered image filename (Filename.Extension)

loadRenderedImage

(boolean) If set to true the croped image will be loaded into Object Image (ObjectName)

useRunNotShellEx

(boolean) This action if set to true, use File.Run command instead Shell.Execute


Return

This action returns a (boolean) value.


ImageMagikEx.CreateGraphicTitle


(boolean) ImageMagikEx.CreateGraphicTitle ( 

(string) imObjectName = "Image1",

(string) imSize = "320x120",

(string) backColor = "transparent",

(string) rightDrawType = "circle",

(string) rightColor = "0000FF",

(string) rightPosition = "250,30",

(string) rightSize = "310,30",

(string) leftDrawType = "circle",

(string) leftColor = "FF0000",

(string) leftPosition = "55,75",

(string) leftSize = "15,80",

(string) fontName = "Calibri",

(number) fontSize = 72,

(boolean) fontDecoration = true,

(string) textFillColor = "00FF00",

(string) strokeTextColor = "000000",

(number) strokeTextSize = 2,

(string) textDecalage = "10,110",

(number) textRotation = -15,

(string) textPosition = "0,0",

(string) textTitle = "ImageMagikEx",

(string) outputFilename = "graphic.png",

(boolean) loadRenderedImage = false,

(boolean) useRunNotShellEx = true )

Description

This action build a graphic title logo image with draws which can be loaded into an object image

Parameters

imObjectName

(string) AMS: Object Image Label ID which contain the rendered title image file

imSize

(string) The size of rendered image title

backColor

(string) The Background image color ( must be an hex color in range of "[000000-FFFFFF]" or "transparent")

rightDrawType

(string) The first geometric figure from the right of the image ( must be "circle" or "rectangle" )

rightColor

(string) The Background of The first geometric figure from the right ( must be an hex color in range of "[000000-FFFFFF]")

rightPosition

(string) The Position of The first geometric figure from the right ( eg: "250,30")

rightSize

(string) The Size of The first geometric figure from the right ( eg: "310,30")

leftDrawType

(string) The first geometric figure from the left of the image ( must be "circle" or "rectangle" )

leftColor

(string) The Background of The first geometric figure from the left ( must be an hex color in range of "[000000-FFFFFF]")

leftPosition

(string) The Position of The first geometric figure from the left ( eg: "55,75")

leftSize

(string) The Size of The first geometric figure from the left ( eg: "15,80")

fontName

(string) The font that will be used to render the text ( All font dont work, you must select an appropriate font - one word only in most of cases )

fontSize

(number) The Size of The font to render the text

fontDecoration

(boolean) The text Decoration ( only UnderLine available at this time )

textFillColor

(string) The fill color of The text ( must be an hex color in range of "[000000-FFFFFF]")

strokeTextColor

(string) The fill color of The text stroke ( must be an hex color in range of "[000000-FFFFFF]")

strokeTextSize

(number) The size of The text stroke

textDecalage

(string) Using this value you can move the text on the image eg: "10,110" ( Assuming the top is: 0,0 but not the top of the global image )

textRotation

(number) The text rotation ( must be in range of [-360/360]

textPosition

(string) The text position on the globale image eg: 0,0

textTitle

(string) The TEXT string that will be rendered eg: "ImageMagikEx"

outputFilename

(string) The output image filename that will be saved in AutoPlay\\Images folder ( A lot of format available: jpg, png, gif, tiff, bmp and more )

loadRenderedImage

(boolean) If this is set to true the rendered image will be loaded into an AMS image object eg: "Image1"

useRunNotShellEx

(boolean) If this is set to true RUN command will be used instead ShellExecute command ( extension .ƒct must be registered with ShellExecute cmd)


Return

This action returns a (boolean) value.
