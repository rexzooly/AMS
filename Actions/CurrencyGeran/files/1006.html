CurrencyGeran.IsCurrency


(boolean) CurrencyGeran.IsCurrency ( 

(string) Value = "EUR 1.000,--" )

Descripcion

Verifica si un valor introducido corresponde a un valor Monetario

Parametros

Value

(string) Representa un valor cualquiera, que se quiera verificar

Obs.: El valor introducido debe de convertirse a un (string), aunque con cualquier valor introducido esta funcion devolvera false. Solamente devolvera true si es introducida un (string) del tipo Currency, es decir de esta forma "EUR 1.000,00" u otro valor similar "$ 1.000,--".


Retorno

Esta accion devuelve un (boolean) como valor.
