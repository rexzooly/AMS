CurrencyGeran.ToNumber


(number) CurrencyGeran.ToNumber ( 

(string) sCurrency = "$ 1.000,--" )

Descripcion

Esta funcion convertira sCurrency, decir una unidad Monetaria en Numero

Ejemplo: $ 1.000,--  Devuelve 1000

Parametros

sCurrency


(string) sCurrency, respresenta un string Ej.: $ 1.000,--

Retorno

Esta accion devuelve (number) como valor



Ejemplos


Ejemplo 1


local nNumber = 1500.15

local sCountry = "$"

local sCurrency = sCountry.." "..nNumber;


local nNumber = CurrencyGeran.ToNumber(sCurrency);



Dialog.Message("ToNumber", "Valor Monetario ingresado: "..sCurrency.."\r\n".."Valor Numerico vuelto: "..nNumber);


Ejemplo 2


local nNumber = "1500,--"

local sCountry = "$"

local sCurrency = sCountry.." "..nNumber;


local nNumber = CurrencyGeran.ToNumber(sCurrency);



Dialog.Message("ToNumber", "Valor Monetario ingresado: "..sCurrency.."\r\n".."Valor Numerico vuelto: "..nNumber);
