CurrencyGeran.GetPorcentCurrency

CurrencyGeran.GetPorcentCurrency


(string) CurrencyGeran.GetPorcentCurrency ( 

(string) sCurrency = " $ 1.000,--",

(string) sCountry = "$",

(boolean) bDecimal = false,

(number) nPorcent = 20,

(boolean) bValue = true )

Descripcion

Dado un valor Monetario cualquiera y un Porcentaje, nos puede devolver el Valor Monetario incrementado en el porcentaje o disminuido. Asi como tambien el porcentaje obtenido.

Parametros

sCurrency


(string) sCurrency = string, que representa una moneda, $ 1.000,00

sCountry

(string) sCountry = string, que reprsenta el simbolo de la moneda, $

bDecimal

(boolean) bDecimal = boolean, que representa un valor boolean indica si suprimen la parte decimal o no.

nPorcent

(number) nPorcent = number, que representa el porcentaje, sobre el sCurrency

bValue

(boolean) bValue = boolean, que indicara si se devolver incrementado o disminuido el sCurrency

Retorno

Esta accion devuelve dos (string) como valor





Ejemplos



Ejemplo 1


local sCurrency = "$ 1500.15";

local sCountry = "$";

local bDecimal = true;

local nPorcent = 20;

local bValue = true


local nCurrency, nPorcent  = CurrencyGeran.GetPorcentCurrency(sCurrency, sCountry, bDecimal, nPorcent, bValue);



Dialog.Message("GetPorcentCurrency", "Valor Monetario a evaluar: "..sCurrency.."\r\n".."Valor Monetario incremtado en Porcentaje "..nPorcent.." : "..nCurrency);


Ejemplo 2


local sCurrency = "$ 1500.--";

local sCountry = "$";

local bDecimal = false;

local nPorcent = 20;

local bValue = false


local nCurrency, nPorcent  = CurrencyGeran.GetPorcentCurrency(sCurrency, sCountry, bDecimal, nPorcent, bValue);



Dialog.Message("GetPorcentCurrency", "Valor Monetario a evaluar: "..sCurrency.."\r\n".."Valor Monetario disminuido en Porcentaje "..nPorcent.." : "..nCurrency);
