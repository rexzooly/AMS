CurrencyGeran.TableCurrency


(Table)  CurrencyGeran.TableCurrency ( 

(string)  sCurrency = "EUR 1.000, -" )

Descripción

Esta accion devuelve una tabla con los valores del tipo de moneda utilizado

Parámetros

sCurrency

(string)  String, representa el valor monetario de un pais determinado.


Retorno

Esta acción devuelve un (table) de valor.


Table

Descripcion

Number

(number). Devuelve un numero del valor monetario

Country

(string). Devuelve  un string que representa el simbolo monetario del pais





Ejemplos



Ejemplo 1


--Valor Monetario


sMoney = "EUR 1.000,--"


Debug.ShowWindow(true);

Debug.Clear();


for j, k in pairs(CurrencyGeran.TableCurrency(sMoney)) do

       

       Debug.Print(j.." = "..k.."\r\n");


end


Ejemplo 2


--Valor Monetario


sMoney = "EUR 1.000,15"


Debug.ShowWindow(true);

Debug.Clear();


for j, k in pairs(CurrencyGeran.TableCurrency(sMoney)) do

       

       Debug.Print(j.." = "..k.."\r\n");


end
