Playlist.AddFile


(number) Playlist.AddFile ( 

(string) Filepath )

Description

Adds an audio file to the playlist and returns the index of the new playlist item, -1 if any error.

Parameters

Filepath

(string) The path to the audio file. Click the browse to select a file.


Return

This action returns a (number) value.


Playlist.AddFolder


(number) Playlist.AddFolder ( 

(string) Folderpath,

(boolean) Recurse,

(function) CallbackFunction,

(function) FileFoundCallbackFunction )

Description

Searches for audio files in a folder and adds them to the playlist, returns the number of tracks added to the playlist, 0 if any error

Parameters

Folderpath

(string) The path to the audio folder.

Recurse

(boolean) Whether to recurse though subfolders when a wildcard is used in the Filename parameter.

CallbackFunction

(function) The name of a function that will be called whenever progress is made in the find operation. (You can use this callback function to display the progress of the find operation in your own custom way.)

FileFoundCallbackFunction

(function) The name of a function that will be called whenever a match is found.


Return

This action returns a (number) value.

Playlist.Load


(boolean) Playlist.Load ( 

(number) Index,

(boolean) PlayAutomatic,

(boolean) Loop )

Description

Loads a playlist item into the audio channel, returns true if the item could be loaded, false if any error.

Parameters

Index

(number) Number index list.

PlayAutomatic

(boolean) Whether to automatically start playing the audio file after it loads.

Loop

(boolean) Whether to make the audio file loop when it reaches the end.


Return

This action returns a (boolean) value.
