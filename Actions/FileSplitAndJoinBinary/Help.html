File.BinarySplit


(string) File.BinarySplit ( 

(string) Source File = "C:\\inputfile.bin",

(string) Dest Folder = "C:\\",

(number) Split Size (KB) = 1000,

(number) Block Size (KB) = 10,

(variant) CallbackFunction = nil,

(boolean) CallbackInSize = false )

Description

Split a binary file into smaller files of the given size

Parameters

Source File

(string) The source file to be splitted

Dest Folder

(string) The output folder to store the splitted files

Split Size (KB)

(number) The split size of output files in KB

Block Size (KB)

(number) The Read/Write Block size, File size must be multiple of Block size.

CallbackFunction

(variant) The fuction to call every block that the function read/wirte.

CallbackInSize

(boolean) Indicates if callback function must be called with size in bytes or percent


Return

This action returns a (string) value.
