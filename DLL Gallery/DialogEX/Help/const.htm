DialogEX Constants

 All the below constants are available in the action editor.
 
 
 Message Type
 
 DEX_OK  OR  0  Shows a "OK" dialog (Default).
 DEX_OKCANCEL  OR  1  Shows a "OK|Cancel" dialog.
 DEX_ABORTRETRYIGNORE  OR  2  Shows a "Abort|Retry|Ignore" dialog.
 DEX_YESNOCANCEL  OR  3  Shows a "Yes|No|Cancel" dialog.
 DEX_YESNO  OR  4  Shows a "Yes|No" dialog.
 DEX_RETRYCANCEL  OR  5  Shows a "Retry|Cancel" dialog.
 
 Dialog Text Align
 
 DEX_ALIGN_LEFT  OR  0  Align dialog text to the left.
 DEX_ALIGN_CENTER  OR  1  Align dialog text to the center.
 DEX_ALIGN_RIGHT  OR  2  Align dialog text to the right.
 
 Dialog Type
 
 DEX_NORM  OR  0  Normal window.
 DEX_TOOL  OR  1  Tool window.
 DEX_NOBORDER  OR  2  Borderless window.
 
 Display type
 
 DEX_SHOW_APP  OR  0  Centers the dialog to the application window.
 DEX_SHOW_SCREEN  OR  1  Centers the dialog to the users screen.
 DEX_SHOW_CUSTOM  OR  2  Shows the dialog at the coords set in X,Y.
 
 Input Types
 
 DEX_INPUT_NORM  OR  0  Normal text input.
 DEX_INPUT_NUMERIC  OR  1  Numeric input, only allows numbers.
 DEX_INPUT_PASSWORD  OR  2  Password input, masks the input eg: ********.
 DEX_INPUT_LOWERCASE  OR  3  Only allow lowercase input.
 DEX_INPUT_UPPERCASE  OR  4  Only allow uppercase input.

 Dialog Animation
 
 DEX_NO  OR  0  No animation
 DEX_FADE  OR  1  Fades the dialog in/out.
 DEX_ROLL_LEFT  OR  2  Rolls the dialog in/out to the left.
 DEX_SLIDE_LEFT  OR  3 Slides the dialog in/out to the left.
 DEX_ROLL_RIGHT  OR  4  Rolls the dialog in/out to the right.
 DEX_SLIDE_RIGHT  OR  5  Slides the dialog in/out to the right.
 DEX_ROLL_TOP  OR  6  Rolls the dialog in/out to the top.
 DEX_SLIDE_TOP  OR  7  Slides the dialog in/out to the top.
 DEX_ROLL_BOTTOM  OR  8  Rolls the dialog in/out to the bottom.
 DEX_SLIDE_BOTTOM  OR  9  Slides the dialog in/out to the bottom.
 DEX_ROLL_TOPLEFT  OR  10  Rolls the dialog in/out to the top left.
 DEX_SLIDE_TOPLEFT  OR  11  Slides the dialog in/out to the top left.
 DEX_ROLL_BOTTOMLEFT  OR  12  Rolls the dialog in/out to the bottom left.
 DEX_SLIDE_BOTTOMLEFT  OR  13  Slides the dialog in/out to the bottom left.
 DEX_ROLL_TOPRIGHT  OR  14  Rolls the dialog in/out to the top right.
 DEX_SLIDE_TOPRIGHT  OR  15  Slides the dialog in/out to the top right.
 DEX_ROLL_BOTTOMRIGHT  OR  16  Rolls the dialog in/out to the bottom right.
 DEX_SLIDE_BOTTOMRIGHT  OR  17  Slides the dialog in/out to the bottom right.
 DEX_CENTER  OR  18  Shows or hides the dialog from its center.
