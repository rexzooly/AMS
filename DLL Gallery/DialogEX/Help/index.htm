DialogEX

 DialogEX is a dll plug-in for AutoPlay Media Studio or any other programming environment that is able to call exported functions from a .dll (dynamic link library)   that will enable you to show a customized dialog window.

 Dialogs:

  • DialogEX.Message()
  • DialogEX.Input()
  • DialogEX.TimedInput()
  • DialogEX.CustomMessage()
  • DialogEX.TimedCustomMessage()
  • DialogEX.IPInput()
  • DialogEX.FileBrowse()
  • DialogEX.FolderBrowse()

 Main Features:

  • Set dialog background and text colors.
  • Set dialog window type (Normal, Tool Window, Borderless Window)
  • Set dialog position, can center to the calling app window, center to the users screen or set a predefined position.
  • Set dialog timeout, if the user takes to long to respond you can close the dialog and continue.
  • Set dialog enter and exit transition
